# Campus Event Tracker

Campus Event Tracker is a Flutter mobile application designed to help students discover, track, and manage events happening on and around their university campus. 

## 🎯 Purpose

The primary goal of this application is to centralize campus activities and make it easier for students to stay engaged with their university community. It solves the common problem of missed opportunities by providing a unified platform for discovering club meetings, guest lectures, sports events, and social gatherings.

## ✨ Features

*   **Discover Events:** Browse a comprehensive list of upcoming events on campus.
*   **Track Favorites:** Save events you're interested in to easily access them later.
*   **Stay Informed:** Get detailed information about event time, location, and organizers.
*   **User Authentication:** Secure login to manage your personal event itinerary.

## 🚀 How to Use

1.  **Sign Up / Log In:** Create an account to personalize your experience.
2.  **Explore:** Navigate through the home feed to discover upcoming events.
3.  **View Details:** Tap on any event card to view the full description, time, and venue.
4.  **Save:** Bookmark events to add them to your personal list.
